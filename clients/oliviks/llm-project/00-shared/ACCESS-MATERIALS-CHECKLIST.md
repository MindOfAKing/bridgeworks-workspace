# Access And Materials Checklist

## Needed From Oliviks

- WordPress/WooCommerce admin access, if current ordering backend stays
- Google Business Profile manager access
- Logo files
- High-resolution food photos
- Founder photos, if they want founder imagery
- Exterior/location photos
- Exact menu list
- Exact product prices
- Exact opening hours
- Direct ordering preference and current order workflow
- WhatsApp number confirmation
- Existing review screenshots or excerpts
- Any press links they want featured
- Exact Foodora listing link

## Needed From Previous Handler

- Namecheap/domain access or DNS owner contact
- WordPress hosting details
- Existing analytics access
- Current site admin username
- Any backups or exported assets

## Do Not Store Here

- Passwords
- API keys
- Client private documents
- Bank/payment credentials
- Raw sensitive exports
