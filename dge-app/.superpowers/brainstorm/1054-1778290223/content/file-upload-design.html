Data flow
1. User selects files or pastes URL
2. Sent to /api/extract
3. Server extracts plain text
4. Text appended to textarea
5. Chip shows filename — click × to remove
6. User edits textarea if needed
7. Hits Analyze as normal
What each format extracts
PDF — full text, page by page
Word (.docx) — paragraphs as plain text
Excel (.xlsx) — each sheet, rows as lines
CSV — rows as comma-separated lines
TXT / MD — raw content
URL — visible page text, tags stripped
