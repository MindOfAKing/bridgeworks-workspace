## Decisions made
- Used Astro SSG (not SSR) for Hostinger compatibility
- Replaced MobileNav React island with vanilla JS inline script (hydration was unreliable)
- Self-hosted logo and hero image instead of external CDNs
- CSS animations + IntersectionObserver instead of Framer Motion (zero library overhead)
- @fontsource for fonts instead of Google Fonts CDN
- Two explicit page files instead of dynamic locale routing
- Stats use 50+ (not 200+) per client preference
- Added scroll-padding-top for fixed nav offset
- Made form status text (Send Another, error messages) bilingual
