## Service Fulfillment Lessons

BridgeWorks should not start AI design from a loose prompt. It should start from a small brand packet, then generate draft assets, then verify by hand and by browser/file output.

## Sales / Positioning Angles

For nontechnical SMBs: one approved brand kit can make future pages, PDFs, decks, and social assets faster without losing consistency.

## Audit Preview Improvements

Add a check for whether the client has usable brand inputs before promising fast AI-generated assets.

## SOP Updates Needed

Create a `Design System Export Gate`:

1. Confirm brand colors, fonts, logo, tone, examples, and do-not-use rules.
2. Save the design system as a reusable local packet.
3. Generate only draft assets first.
4. Verify copy, claims, layout, accessibility, responsiveness, and export format.
5. Move to code only after approval.
6. No new connector, paid tool, GitHub install, or client data upload without Emmanuel approval.
