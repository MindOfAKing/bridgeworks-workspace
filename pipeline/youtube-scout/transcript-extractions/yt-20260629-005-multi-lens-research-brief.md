## Systems And Processes

BridgeWorks version:

1. Define a narrow research question.
2. Run five lenses: operator, buyer, skeptic, finance, market context.
3. Require each lens to cite sources or state uncertainty.
4. Create a contradiction map.
5. Synthesize only claims that survive review.
6. Mark claims as confirmed, corrected, demoted, or needs source.
7. Convert only the surviving insight into a BridgeWorks action.
