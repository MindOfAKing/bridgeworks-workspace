## Problem

AI chat creates answers, but the working knowledge base stays separate from the files, workflows, and recurring business tasks. That means context does not compound.

## Framework

Build an operating folder with:

1. A top-level instruction file.
2. A memory log.
3. Workflow subfolders.
4. Workflow-specific instruction files.
5. Skills for repeatable actions.
6. Connectors for Gmail, Calendar, Drive, and other tools.
7. Scheduled tasks for repeatable workflows.

## Tool / stack

Claude Cowork, local folders, CLAUDE.md-style instruction files, MEMORY.md-style logs, Obsidian, Gmail connector, Google Calendar connector, Google Drive connector, skills, scheduled tasks, mobile dispatch.

## Workflow

1. Create a parent OS folder.
2. Generate the top-level instruction and memory files.
3. Add subfolders only for repeatable workflows, not random categories.
4. Connect required tools.
5. Let Cowork inspect real inputs, such as recent email, to build categories and tone rules.
6. Save working routines as skills.
7. Schedule routines only after they work manually.
8. Keep a local limitation note: scheduled work depends on the computer and app being open.
