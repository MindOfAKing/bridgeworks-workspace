## BridgeWorks adaptation

Create a no-spend `AI Visibility Foundation QA` row for all Foundation websites and GEO audits:

1. Robots and crawler access check.
2. JavaScript visibility check for key page content.
3. Organization / LocalBusiness / FAQ schema where appropriate.
4. Direct answer block in the first third of important pages.
5. FAQ section based on real customer questions.
6. Comparison or service-fit table only where truthful.
7. Review request guidance that asks for specific outcomes, not fake praise.
8. External mention map: YouTube, LinkedIn, directories, industry pages, but no spam.
