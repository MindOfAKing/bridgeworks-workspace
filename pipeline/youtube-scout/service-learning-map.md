## Scoring For Videos

Score each video from 0 to 20.

| Dimension | Points |
|---|---:|
| Directly improves a BridgeWorks service | 0 to 5 |
| Can become an SOP/checklist/template | 0 to 4 |
| Can improve sales or client delivery this month | 0 to 4 |
| Specific and practical, not motivational | 0 to 3 |
| Fits current business priority | 0 to 2 |
| Can become content/proof | 0 to 2 |

Decision:

- 16 to 20: Extract this week.
- 11 to 15: Queue for later.
- 6 to 10: Save only if highly relevant.
- 0 to 5: Skip.
