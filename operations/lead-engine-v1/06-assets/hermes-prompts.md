
## Run a Lead Leak Review

```text

```

## Create weekly lead review

```text
Use the prospect tracker and this week's notes. Generate the BridgeWorks Weekly Lead Review. Include numbers, best opportunities, FWT filter, uncomfortable truth, and next week's top 5 actions. Keep the system embarrassingly simple: find 10 businesses, score them, message 3, follow up, book calls, repeat weekly.
```
